<h2 align="center"> Módulos del sistema y su descripción  </h2>

<h3> 1. SEGURIDAD Y USUARIOS </h3>
<ul>
  <li> 1.1 Login y Autenticación:</li>
    <ul><li>Manejo de acceso de operadores y administradores.</li></ul> <br>
  <li> 1.2 Administración de operadores:</li>
    <ul> <li>Gestión de las cuentas de usuario que generan los reportes. </li></ul> <br>
  <li> 1.3 Roles y Permisos:</li>
    <ul> <li> Definición granular de qué roles pueden crear, ver, editar, y eliminar reportes.  </li> </ul>
</ul>

<h3> 2. CATALOGO </h3>
<ul>
  <li>2.1 Administración de Catálogos</li>
  <ul><li>Gestión de listas predefinidas usadas en el formulario (ej. Tipos de Dispositivo, Ubicaciones, Tipos de Falla).</li></ul>
</ul>

<h3> 3. FLUJO DE REPORTES </h3>
<ul>
  <li> 3.1 Interface de Operador</li>  
  <ul><li>El formulario central para la alta de nuevos reportes, incluyendo la subida de fotos/videos, adicional opciones para <b> consulta , modificacion y eliminiación </b> de reportes previos (en caso de contar con privilegios)</li></ul>
</ul>

<h3> 4. Modulos Auxiliaries </h3>
<ul>
  <li> 4.1 Configuración del Sistema</li>  
  <ul><li>Parámetros generales del sistema (ej. límites de tamaño de archivo, zonas horarias, configuraciones de correo para notificaciones).</li></ul>
</ul>
